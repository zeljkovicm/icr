# Interakcija čovek–računar 2025

Izvorni kod aplikacije sa vežbi iz predmeta Interakcija čovek–računar na 4. godini Tehničkog fakulteta Univerziteta Singidunum.

Aplikacija je razvijena koristeći sledeće tehnologije:
- Angular 20+
- Bootstrap 5.3
- Font Awesome 7
- Axios HTTP Client
- Rasa Open Source

## Pokretanje projekta

Da biste pokrenuli projekat, potrebno je da imate instaliran Node na vašem računaru.
Aplikaciju možete pokrenuti pomoću komande: `ng serve`
